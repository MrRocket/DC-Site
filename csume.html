C.S.U.M.E. for Doom Connector

 

What the heck is C.S.U.M.E?

C.S.U.M.E is a (Client Server Universal Machine Emulator)
Also known as CSUME: it's based on some previous source releases of MAME and UME.

CSUME was created by: MisterTea

CSUME is a branch of UME
MisterTea's github: https://github.com/MisterTea/MAMEHub

While it's ROM support may not be as vast as the current versions of MAME or UME, this emulator has the ability to host a game over the net or LAN, using a TCP/IP Peer to Peer based connection.
From commandline or by means of a client based chat game launcher like MAMEHub.

View the Wiki for more info about MAMEHub and CSUME:
https://github.com/MisterTea/MAMEHub/wiki

===========================================================
Why CSUME in Doom Connector though?

The Doom Connector Team took specail interest in CSUME sense it has the ability to play games over the Internet. And while Doom Connector mainly specaillzes in DooM based mutliplayer game support, it also now supports CSUME, we figured, why not add some support for original arcade games also? If the emulator has the feature, why not add it?

Doom Connector is now ready for internet ARCADE GAME action!
